# 江苏泓基金属材料有限公司官网 - 详细需求文档

## 项目概述

参考网站：
http://www.dtyuanyang.com/en/

复刻这个网站的前端页面。

### 1.1 项目背景

江苏泓基金属材料有限公司（以下简称"泓基金属"）是一家专注于不锈钢管配件生产的企业。为提升企业形象、拓展线上业务，需要建设一个功能完善的中英文双语官网，并配套后台管理系统。

### 1.2 项目目标

- 建立专业、现代化的企业形象展示平台
- 实现产品信息的在线展示与管理
- 提供新闻动态发布功能
- 支持中英文双语切换
- 建立后台管理系统，实现内容自主维护

### 1.3 目标用户

- **前端用户**：国内外潜在客户、合作伙伴、行业访客
- **后台用户**：企业管理员、内容编辑、产品管理员

## 4. 产品管理模块（核心）

### 4.2 产品管理

#### 4.2.1 产品列表

| 功能     | 描述                                 | 优先级 |
| -------- | ------------------------------------ | ------ |
| 列表展示 | 缩略图、名称、分类、价格、库存、状态 | P0     |
| 搜索筛选 | 按名称、分类、状态筛选               | P0     |
| 批量操作 | 批量删除、批量上下架                 | P1     |
| 快速编辑 | 弹窗快速修改价格、库存               | P1     |
| 导出导入 | Excel导入导出                        | P2     |

#### 4.2.2 产品添加/编辑

**基础信息**：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 产品名称 | 文本 | 是 | 中英文 |
| 产品副标题 | 文本 | 否 | 简短描述 |
| 产品分类 | 选择 | 是 | 关联分类 |
| 产品图片 | 图片组 | 是 | 最多10张，支持拖拽排序 |
| 产品视频 | 视频 | 否 | 产品介绍视频 |
| 产品状态 | 开关 | 是 | 上架/下架 |

**规格参数**（动态规格）：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 口径范围 | 文本 | 否 | 如：DN15-DN3000 |
| 压力等级 | 文本 | 否 | 如：CL75-CL2500 |
| 材质 | 多选 | 否 | 碳钢、不锈钢、双相钢等 |
| 执行标准 | 多选 | 否 | GB、HG、ISO、ANSI等 |
| 重量 | 数值 | 否 | 单位：kg |
| 产地 | 文本 | 否 | |

**详细描述**：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 产品详情 | 富文本 | 否 | 图文详情，中英文 |
| 技术参数表 | 表格 | 否 | 规格参数表格 |
| 应用场景 | 富文本 | 否 | 应用案例描述 |
| 包装运输 | 富文本 | 否 | 包装方式说明 |
